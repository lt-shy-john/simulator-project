## Introduction
At `/simulations` page, you will see the following screen: 
![A webpage with a big table in the middle.](/docs/Simulation.PNG)

| Name             | Identifier                                            |
|------------------|-------------------------------------------------------|
| Creation Date    | Creation date of the simulation                       |
| Status           | Can be "ready", "in progress"                         |
| Action           | Actions on the simulation set, see below              |
| Number of Agents | (hidden) Number of (initial) agents in the simulation |
| Simulation Time  | (hidden) Total simulation time                        |

### Action
By clicking on the 3 ellipses, you can see the following options: 
* "Run" will start the simulation run
* "View" views the simulation settings
* "Settings" changes settings
* "Delete" will delete the simulation set

### Create Simulation
1. Click "Create"
2. Populate the settings
3. Click "Next"
4. Confirm the details and click "Create"
