## Getting Started

### Installation
You need to install the following libraries
* `django`
* `django-cors-headers`
* `djangorestframework`
* `drf-spectacular`

## Testing
At the root of the Simulation Project Backend folder, run `py manage.py test`. 

To export the test results, you can run `py manage.py test > test_result_{date}.txt`.
